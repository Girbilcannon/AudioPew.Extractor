# AudioPew.Extractor
Just a simple audio extractor GUI that uses ffmpeg for high quality batch audio extractions from videos. Because for some reason it's hard to find free tools that do this.

FFMPEG comes pre-zipped for convenience, but if you are compiling this project yourself or you are updating to a later FFMPEG version, You will need the latest FFMPEG release. Just simply drag the contents of the bin folder into the AudioPewExtractor "Tools" folder (except for ffplay.exe....you can toss that).
